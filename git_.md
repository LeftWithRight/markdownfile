##  分支管理：
  
  
可以使用以下命令创建、切换和合并分支：  
```
git branch <branchname>        # 创建新分支
git checkout <branchname>      # 切换到指定分支
git merge <branchname>         # 合并指定分支到当前分支  
 ```
  
## 远程仓库管理：
可以使用以下命令与远程仓库进行交互：
```
git clone <url>                # 从远程仓库克隆
git pull                       # 从远程仓库拉取最新代码到本地
git push                       # 将本地代码推送到远程仓库  
```
